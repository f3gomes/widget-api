FeedbackAPI Documentation

API Version: 1.0.0

POST /api/feedback/new

Parameters

  • type (required) : string - Type of feedback ("BUG", "IDEA", "OTHER")
  • comment (required) : string - Feedback description
  • companyId (required) : string - ID of company

Response

201 Created

                {
                  "feedback": {
                    "id": 4,
                    "type": "BUG",
                    "comment": "New bug found",
                    "screenshotUrl": null,
                    "createdAt": "2024-10-23T01:25:30.677Z",
                    "companyId": "91af855c-0f82-4294-b8a5-8ffccb42b8df"
                  }
                }
POST /api/company/new

Parameters

  • name (required) : string - Company name
  • cnpj (optional) : string - CNPJ Number
  • address (optional) : string - Company address
  • phone (optional) : string - Company contact number
  • email (optional) : string - Company email address

Response
